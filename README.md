# micro-dotfiles
Dotfiles for XMonad &amp; XMobar

## Dependencies
### Main
- xmonad
- xmobar
- alacritty
- dmenu

### Optional
- fish
- maim
- xclip
- neofetch

## Installation
```shell
git clone https://github.com/Ad4ndi/micro-dotfiles
cp -r micro-dotfiles/.xmonad ~
cp -r micro-dotfiles/.config/* ~/.config/
xmonad --recompile && xmonad --restart
rm -rf micro-dotfiles
```
